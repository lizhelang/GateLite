# GateLite Logo Assets

The canonical GateLite logo files live in `public/brand/gatelite/` so the web
app, browser favicon, web manifest, and documentation all reference project-local
assets.

- `icon-*` files are square app, favicon, avatar, and sidebar marks.
- `horizontal-*` files are wordmark lockups for documentation and larger brand
  placements.
- `light` and `dark` variants are selected by the app theme.
- `mono` variants are available for single-color usage.
